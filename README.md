
# Need for Speed: Most Wanted (2012) Discord Rich Presence

A lightweight C++ ASI plugin that expands Discord Rich Presence into Need for Speed: Most Wanted (2012).

## Features
* **Dynamic Content**: Displays your current car, game mode, and active pursuit score (Speed Points).
* **Smart Localization**: Automatically toggles between English and Russian based on your Windows system display language.
* **Open Source**: Full source code provided to ensure safety and transparency.

## Installation
Place the compiled `.asi` file into your game's `scripts` folder,  and `.dll` files into game root directory
